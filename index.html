    Albatroz

    Albatroz is a web system developed for a dental clinic that seeks to assist in the management and control of appointment schedules, clients and the queue for assistance.

    The application front end was developed using React.js and CSS. In addition to libraries like the React Router Dom to configure routes, Axios to consume APIs and other libraries like SweetAlert2 and React icons.

    It was an excellent opportunity to implement user authentication with jwt for the administrator's private routes.
